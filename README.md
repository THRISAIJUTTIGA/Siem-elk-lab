# SIEM Lab - ELK Stack Threat Detection

## Project Overview
Built a mini SOC analyst lab using ELK Stack to detect simulated cyber attacks.

## Tools Used
- Ubuntu 22.04 Server (VM)
- Elasticsearch 8.x
- Kibana 8.x
- Filebeat

## What I Did
- Set up Ubuntu VM using UTM on MacBook M2
- Installed and configured ELK Stack
- Collected system logs using Filebeat
- Simulated SSH brute force attack
- Detected attack in Kibana dashboard
